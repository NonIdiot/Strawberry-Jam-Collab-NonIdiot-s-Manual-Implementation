This is NonIdiot's Archipelago Manual for Celeste: Strawberry Jam Collab.

To start:
- Load this APWorld like normal.
- Start a new save file for Celeste with the Strawberry Jam Collab mod installed (with the Strawberry Jam Collab campaign selected when making the new save, for convenience of not having to beat vanilla's Prologue too).
- Starting either at the start of Beginner Lobby or at the start of Prologue (depending on your YAML settings), your movement becomes restricted.

Some basic information to know:
- You can restrict the movement you need to with the submenus of the Extended Variants option in the pause menu (specifically under the Movement submenu). Disabling neutral jumps requires you to set "Disable neutral jumping" to on, disabling grabs requires you to set "True No Grabbing" to on, and disabling different dash directions requires you to set "Dash Direction" to Custom, and then set each individual dash direction in the "Allowed Dash Directions" sub-option to off.
- As mentioned in the YAML, your dash/grab/neutraljump items give you abilities in different ways depending on how the options are set. If Dashsanity is on, each dash direction is restricted individually; But otherwise if it is off the Dash item gives all dash directions at once. If Progressive Grabbing is on, the first Progressive Grabbing item allows you to neutral jump, and upon obtaining both, you are allowed to neutral jump and grab; But if it is off, the Neutral Jump item only allows neutral jumping and the Grab item only allows grabbing.
- A lot of logic includes routing challenges that go up to Expert Difficulty (no matter what the current lobby you are in is). This means you will have to come up with creative ways to work around your lack of certain abilities. However, this works much better in multiworlds rather than single-world playthroughs, as Archipelago tends to give you movement abilities much earlier in progression. Also to note, some entities may act strangely or incorrectly with restricted dash directions (such as bubbles/boosters, especially those in Sea of Soup). These are, for the most part, accounted for in logic.
- While pixel-perfect gaps and demodashes (using the Crouch Dash keybind) ARE considered in logic, demodashing through spinners are almost entirely not accounted for in logic. This may allow for out-of-logic skips.
- If the Region Locking option is on, you will need unique keys to go through certain colored-on-map regions named by the bench that they contain. You cannot go through a region you do not have unlocked to get to a region that you do have unlocked. If you need to check if you are able to be within a region, press TAB while on the ground in order to see if the ground you are on is of the color of a region you have unlocked.
- Logic does not take into account backtracking, so you may need to use Restart Chapter now and then. This includes in the Lobbies themselves, too (as doing so brings you to their default spawn point). In fact, you can't get all Sphere 1 locations without using Restart Chapter at least once (if Skip Prologue is true).
- You may use the Debug Map (press F6) to warp past the Heart Gate in a lobby if you have enough of the lobby's Mini-Hearts in the Archipelago. However, it is discouraged to use for any purposes other than this.
- If you have Deathlink on, you are in track of when you send one. In regular Celeste (Open World), there is an amnesty system that makes it so that each X amount of deaths sends a single deathlink. I would personally recommend installing the Death Tracker mod and choosing an amnesty number that is easily divisble (such as 10, 20, 25, 50, etc) to make cause a death. There is no YAML option for amnesty, so tracking this is entirely on the player.

Additional things to know, due to this being a work in progress:
- Only Beginner Lobby is supported currently, however it is planned to include at least Intermediate and Advanced Lobbies too.
- Not all out-of-logic stuff is inherently a bug. If you get an item before you're supposed to in-logic without ignoring keys or movement restrictions, you can press F1 in the Manual client to allow out-of-logic location checking. Most of the out-of-logic stuff is due to either silly oversight, or just doing a trick too many times and giving up to assume that the trick is Low Grandmaster or higher.
